## DataMed

# Run the application - STEPS:

Clone the repository

```bash
Project repo: https://github.com/natashaejercito/data-med
```

Create a conda environment after opening the repo

```bash
conda create -n datamed python=3.10 -y
```

```bash
conda activate datamed
```

Install the requirements

```bash
pip install -r requirements.txt
```
